# Juju Recipe

A simple, visually appealing Android app to manage your recipes and generate grocery lists.

## Features
- **Recipe Management**: Add, edit, and delete your favorite recipes.
- **Dynamic Grocery List**: Select recipes to automatically generate a consolidated shopping list.
- **Custom Culinary Theme**: A warm, Material 3-based design with "Culinary" tones (Oranges/Earthy).
- **Ingredient Customization**: Override ingredient counts per recipe or per ingredient for flexible shopping.
- **Modern Navigation**: Fluid slide and fade transitions between screens.

## Tech Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose (Material 3)
- **Database**: Room Persistence Library
- **Navigation**: Jetpack Compose Navigation
- **Architecture**: MVVM
